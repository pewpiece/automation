# Automation Projects

This repository contains a collection of small automation exercises and utilities.

## Projects

- `project1_rename/` - rename downloaded files using a pattern-based workflow
- `project2_mover/` - organize mixed file types into separate folders
- `project3_api/` - API-driven data cleanup and country data processing
- `project3_csv/` - CSV cleaning, combining, and reporting utilities
- `project4_backup/` - backup system tooling
- `project5_ml/` - machine learning training and prediction scripts
- `project6_api/` - API-related helper scripts
- `project7_prefect/` - Prefect pipeline and run reports

## Notes

- Generated caches and local virtual environments are ignored via `.gitignore`.
- The repository is intended as a practice workspace for Python automation tasks.
